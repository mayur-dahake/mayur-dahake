### [Mayvio UI](https://github.com/mayur-dahake/mayvio-ui)

**Stack:** TypeScript, JavaScript, HTML, CSS  
**Live:** [mdevhubui.vercel.app](https://mdevhubui.vercel.app)

### [portfolio-v5](https://github.com/mayur-dahake/portfolio-v5)
Portfolio monorepo with isolated frontend and backend apps, using shared types and a managed PostgreSQL workflow through Prisma and Supabase.

**Stack:** React, TypeScript, Express, Prisma, PostgreSQL  
**Live:** [portfolio-v5-frontend.vercel.app](https://portfolio-v5-frontend.vercel.app)

### [MDevHub UI Library](https://github.com/mayur-dahake/mdevhub-ui-lib)
Angular UI component library focused on reusable, customizable components such as paginators and file-upload flows, built on Angular Material.

**Stack:** Angular, TypeScript, Angular Material

Angular file upload component with drag and drop, URL upload, preview support, validation rules, and progress states for product-grade upload workflows.

**Stack:** Angular, TypeScript, SCSS

### [custom-paginations](https://github.com/mayur-dahake/custom-paginations)
Reusable Angular pagination component published with a live demo and designed for integration into broader UI workflows.

**Stack:** TypeScript  
**Live:** [custom-paginations.vercel.app](https://custom-paginations.vercel.app)

### [currency-converter-app](https://github.com/mayur-dahake/currency-converter-app)
